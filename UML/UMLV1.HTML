Estructura UML: Servicio Panel Jefatura y Servicios Generales

Flujo Técnico de Datos en Tiempo Real e Integración de Angular con Supabase Backend.

sequenceDiagram
    autonumber
    actor Jefe as Jefe de Seguridad / Servicios Generales
    participant Angular as Angular Frontend (LayoutDashboard)
    participant State as State Controller (RxJS / NgRx / Services)
    participant Supabase as Supabase Backend (PostgreSQL)
    participant Realtime as Supabase Realtime (WebSockets)
    participant GuardApp as App Guardia / Conductor

    %% --- BLOQUE DE CONTROL DE ACCESO / AUTENTICACIÓN ---
    Note over Jefe, Supabase: Inicialización de Sesión Operativa
    Jefe->>Angular: Ingresa a "Panel Jefatura y Servicios Generales"
    Angular->>Supabase: Validar Rol de Usuario (Firebase Auth / Supabase Auth)
    Supabase-->>Angular: Token Válido (Role: 'jefatura' | 'servicios_generales')

    %% --- FLUJO A: GESTIÓN DE REQUISITOS DE BLOQUEOS (RF-3.1) ---
    Note over Jefe, GuardApp: FLUJO A: Gestión de Reservas y Bloqueos (RF-3.1)
    Jefe->>Angular: Hace clic en espacio vacío en Mapa (2D SVG / 3D Three.js)
    Angular->>State: selectSpace(spaceId)
    State-->>Angular: Actualiza selectedSpaceId y abre SidebarControl (Drawer)
    Jefe->>Angular: Introduce motivo ("Mantenimiento" / "Visita VIP") y presiona "Bloquear"
    Angular->>State: handleBlockSpace(spaceId, reason)
    
    activate State
    State->>Supabase: HTTP PATCH /rpc/block_parking_space {id: spaceId, status: 'BLOCKED', blockReason: reason}
    activate Supabase
    Supabase->>Supabase: Registra en BD & Inserta EventLog (type: 'security_block')
    Supabase-->>State: 200 OK (Confirmación de Persistencia)
    deactivate Supabase
    
    State->>Realtime: Broadcast evento 'SPACE_UPDATED' {id: spaceId, status: 'BLOCKED'}
    deactivate State
    
    par Notificación en Tiempo Real (<500ms)
        Realtime-->>Angular: WebSocket: Actualización de Estado (Update Local State)
        Angular-->>Jefe: Renderiza espacio en color Gris/Bloqueado en el Mapa
    and Sincronización con otros Servicios
        Realtime-->>GuardApp: WebSocket: Forzar desocupación / Inhabilitar Selección
        Note over GuardApp: Espacio bloqueado para Guardias y Conductores
    end

    %% --- FLUJO B: DASHBOARD DE ESTADÍSTICAS (RF-3.2) ---
    Note over Jefe, Supabase: FLUJO B: Dashboard de Estadísticas y Tendencias (RF-3.2)
    Jefe->>Angular: Selecciona pestaña "Métricas de Sede"
    Angular->>State: generateDashboardReport()
    
    activate State
    State->>Supabase: HTTP GET /v1/parking_stats & /v1/history_logs
    activate Supabase
    Supabase->>Supabase: Calcular métricas (occupancyRate, peakHours, blockedTotal)
    Supabase-->>State: Retorna JSON (Modelo ParkingStats + Logs Históricos)
    deactivate Supabase
    
    State-->>Angular: Inyecta datos a componentes reactivos (KPIWidgets / Recharts)
    deactivate State
    Angular-->>Jefe: Despliega Gráficos de Ocupación, Slots PMR/EV y Alertas en tiempo real
